# MRB STUDIO

Современный корпоративный сайт для IT-студии **MRB STUDIO**, специализирующейся на разработке Telegram-ботов, Mini Apps, веб-сайтов и парсеров.

![MRB Studio Preview](/public/img/1.gif)

## 🎨 Особенности дизайна
*   **Visual Style**: Эффектный темный дизайн в стилистике Neon / Cyberpunk (вдохновлено GTA VI).
*   **Animations**: Плавные микро-взаимодействия, скролл-анимации и переходы (`framer-motion`).
*   **3D Effects**: Атмосферный фон с эффектом темной воды на главной странице (`three.js`).
*   **Glassmorphism**: Стеклянные карточки с неоновыми границами.
*   **Responsive**: Полная адаптация под мобильные устройства и десктопы.

## 🛠 Технологический стек
*   **Framework**: [Next.js](https://nextjs.org/) (App Router, TypeScript)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Animations**: [Framer Motion](https://www.framer.com/motion/)
*   **3D Graphics**: [Three.js](https://threejs.org/) (React Three Fiber)
*   **Deployment**: Vercel / GitHub Pages

## 📂 Структура проекта
*   `/app` - Основная логика приложения и страницы.
*   `/components` - Переиспользуемые UI компоненты (Header, Footer, Cards).
*   `/public` - Статические файлы (изображения, шрифты).

## 🚀 Запуск локально

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/bag1-bot/mrbstudio_site.git
    ```
2.  Установите зависимости:
    ```bash
    npm install
    # или
    yarn install
    ```
3.  Запустите сервер разработки:
    ```bash
    npm run dev
    ```
4.  Откройте [http://localhost:3000](http://localhost:3000) в браузере.
